SKU Image Sync

Aplicativo para sincronizar imagens de produtos da Nuvemshop usando o SKU como chave.

Status do backend

Use /health para verificar o servidor e /auth/status para testar a conexao com a API da Nuvemshop.

Rotas principais

  • POST /sync/dry-run simula a sincronizacao.
  • POST /sync/add adiciona imagens novas.
  • POST /sync/sync sincroniza por hash local.
  • POST /sync/replace remove imagens antigas e reenvia tudo.
