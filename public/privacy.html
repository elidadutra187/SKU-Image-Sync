Politica de Privacidade

Esta Politica de Privacidade descreve como o SKU Image Sync coleta, usa e protege suas informacoes.

1. Informacoes Coletadas

O SKU Image Sync acessa os seguintes dados da sua loja Nuvemshop:

  • Informacoes de produtos (ID, nome, SKU)
  • Imagens de produtos
  • ID da loja

2. Uso das Informacoes

As informacoes sao utilizadas exclusivamente para:

  • Identificar produtos pelo SKU
  • Sincronizar imagens entre suas pastas locais e a Nuvemshop
  • Gerar relatorios de sincronizacao

3. Armazenamento

O aplicativo armazena localmente:

  • Arquivo de estado para controle de sincronizacao
  • Relatorios CSV das operacoes realizadas
  • Token de acesso no arquivo .env (configurado por voce)

Nenhum dado e enviado para servidores de terceiros alem da API oficial da Nuvemshop.

4. Seguranca

Recomendamos:

  • Manter o arquivo .env seguro e nunca compartilha-lo
  • Usar tokens com permissoes minimas necessarias
  • Nao versionar arquivos .env no Git

5. Compartilhamento

Nao compartilhamos suas informacoes com terceiros. Todos os dados permanecem entre sua maquina local e a API da Nuvemshop.

6. Seus Direitos

Voce pode a qualquer momento:

  • Revogar o acesso do aplicativo na Nuvemshop
  • Excluir os arquivos locais de estado e relatorios
  • Solicitar informacoes sobre os dados processados

7. Contato

Para duvidas sobre esta politica, entre em contato atraves da pagina de Suporte.
